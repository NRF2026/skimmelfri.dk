# skimmelfri.dk

This repository contains the development of an informational Danish website about mold prevention and indoor climate.

## Project goal
Build a trustworthy, non-commercial informational website based on specification.md.

The website must:
- feel professional and trustworthy
- not appear commercial
- be written in Danish
- follow Danish authoritative sources
- comply with GDPR and cookie regulations

## Instructions for Claude Code
1. Read specification.md completely before making changes.
2. Create a clean, minimal, professional website.
3. Prefer static site architecture.
4. Use semantic HTML and accessible design.
5. Focus first on structure, layout, and navigation.
6. Do NOT add marketing language or monetization features.
7. Implement GDPR-compliant cookie handling as described.

Start by scaffolding the full website structure.
